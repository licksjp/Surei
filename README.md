### ■●数霊(かずたま)について
### ・「数霊」は、存在する万物を表現する素になっている。
### 『地球上の万象万物は、宇宙全体の生成化育の原理から
### 「数霊」と「言霊」として表現され、万象へと化成する。
### それが、直ちに万宥の生命である。』
### ・音と色を波長で分析すると、数で表現される。
### ・辺の数や角度は数を用いて説明される。
### ・宇宙空間の一切のものはすべて、数で表現され、
### 万物は数から成り立っている。（by ピタゴラス)
